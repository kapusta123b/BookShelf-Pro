1 About This Project

BookShelf Pro is a portfolio project — built to learn and demonstrate full-stack web development with Django. It is not a real service, has no commercial intent, and carries no legal obligations.

The project integrates Django, PostgreSQL, SCSS, JavaScript, and the Open Library API. The source code is open on GitHub and is the main thing this page is meant to showcase.

2 Demo Use

Do whatever you want!

3 Content & Data

All book metadata displayed in the catalog — titles, authors, covers, subjects — comes from the Open Library API, operated by the Internet Archive. That data is published under a CC BY 4.0 licence.

This project does not own or claim any rights over book metadata. All credit for that data goes to the Open Library community and contributors.

4 Source Code

The BookShelf Pro source code is published under the MIT licence. You are free to read it, learn from it, fork it, or adapt it for your own projects.

If you find a bug or have a suggestion, feel free to open an issue on GitHub — always appreciated.
