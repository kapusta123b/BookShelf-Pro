1 About This Project

BookShelf Pro is a portfolio project built to demonstrate full-stack Django development skills. It is not a commercial product and is not intended for real-world use.

The project is open-source and publicly available on GitHub. It was built as a learning exercise and showcase piece, integrating Django, PostgreSQL, SCSS, and the Open Library API.

If you are reviewing this project as part of evaluating my work — thanks for taking the time to look around!

2 What Gets Stored

If you create an account or log in (e.g. to test the features), the following data is saved to the database:

  • Username and email address
  • Hashed password (never plain text — Django's PBKDF2)
  • Any books, statuses, ratings, or reviews you add
  • Standard Django session data

This data is stored only to make the app functional during your session. No analytics, tracking, or advertising data is collected.

Since this is a demo environment, please do not use real personal information or passwords you use elsewhere.

3 Third-Party Services

The project uses a few external services:

  • Open Library API — book metadata (titles, authors, covers) is fetched from openlibrary.org. No personal data is sent there.
  • Google / Facebook OAuth — optional social login via django-allauth. Only your basic profile info (name, email) is received with your consent.
  • Google reCAPTCHA v2 — shown on auth forms. Google's own privacy policy governs reCAPTCHA interactions.
  • Google Fonts — DM Sans is loaded from Google's CDN. Google may log your IP as part of that request.

4 Cookies & Storage

Only the minimum necessary cookies are used:

  • Session cookie — keeps you logged in
  • CSRF token — standard Django security cookie for form submissions
  • localStorage — used client-side only to remember sidebar and scroll state; nothing is sent to the server

There are no analytics cookies, advertising cookies, or any third-party tracking scripts.

5 Source Code

The full source code for this project is available on GitHub under the MIT licence. Feel free to explore the implementation, open issues, or fork it.
